前言

MySQL Binary Log也就是常说的bin-log,是mysql执行改动产生的二进制日志文件,其主要作用有两个:

  • 数据恢复
  • 主从数据库。用于slave端执行增删改，保持与master同步。

准备工作(开启binary log功能)

  • 修改mysql的配置文件,添加一句log_bin = mysql_bin即可
